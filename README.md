simply-vasttrafik
=================

A commute application for Västtrafik, made for the Pebble smartwatch. Uses simply.js for functionality.

Set up
------
1. Clone this repo
2. Create an account and recive a apikey at http://labs.vasttrafik.se/.
3. Create a environment.js, matching environment.js.template and place your apikey in environment.vasttrafikapikey.
4. Place simply-vasttrak and environment.js on a webhost.
5. Install Simply.js on your Pebble.
6. Open Simply.js configuration on your phone, and write your webaddress in the field
7. You are ready to go!
